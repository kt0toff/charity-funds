# 🎮 Віддалене Керування ПК

## Що додано:

### 1. **Трансляція екрану в реальному часі**
- Стрімінг екрану через FFmpeg
- Якість 720p, 15 FPS для оптимальної швидкості
- Автоматичне перепідключення при втраті зв'язку

### 2. **Повне керування ПК**
- **Миша**: Тачпад, клік лівою/правою кнопкою, скрол
- **Клавіатура**: Введення тексту, швидкі клавіші (Enter, Esc, Ctrl+C/V, Alt+Tab)
- **Медіа**: Play/Pause, Next, Previous, Volume Up/Down, Mute
- **Система**: Робота/Сон режими, блокування, термінал, вимкнення/перезавантаження

### 3. **Веб-інтерфейс**
- Адаптивний дизайн для смартфонів та планшетів
- Красивий градієнтний інтерфейс
- Індикатор статусу трансляції (LIVE/Offline)

## Як запустити:

### 1. Встановіть залежності (якщо ще не встановлені):

```bash
cd ~/Documents/bot
source venv/bin/activate
pip install flask flask-cors opencv-python psutil python-dotenv requests aiogram
```

### 2. Встановіть FFmpeg (для трансляції екрану):

```bash
sudo dnf install ffmpeg xdotool -y
```

### 3. Налаштуйте .env файл:

```bash
nano ~/Documents/bot/.env
```

Додайте або перевірте:
```
WEB_APP_URL=http://localhost:5000
PC_PASSWORD=3451
PC_BOT_TOKEN=ваш_токен
PC_USER_ID=ваш_id
```

### 4. Запустіть сервер:

```bash
cd ~/Documents/bot
python app.py
```

### 5. Запустіть Telegram бота (в іншому терміналі):

```bash
cd ~/Documents/bot/pc_remote_control
python remote_bot.py
```

## Як користуватися:

### Через Telegram:
1. Відкрийте бота
2. Натисніть кнопку **"🎮 Віддалене Керування"**
3. Відкриється веб-інтерфейс з трансляцією екрану

### Через браузер напряму:
1. Відкрийте: `http://localhost:5000/remote`
2. Або з іншого пристрою в локальній мережі: `http://IP_ВАШОГО_ПК:5000/remote`

## Функції веб-інтерфейсу:

### 🖱️ Миша:
- **Тачпад**: Водіть пальцем/мишею по чорній області для переміщення курсора
- **Лівий/Правий клік**: Кнопки під тачпадом
- **Скрол**: Кнопки ⬆️ Вгору / ⬇️ Вниз

### ⌨️ Клавіатура:
- **Поле вводу**: Введіть текст і натисніть "📤 Відправити"
- **Швидкі клавіші**:
  - ↵ Enter
  - ⎋ Esc
  - ⌫ Backspace
  - 📋 Ctrl+C (Copy)
  - 📄 Ctrl+V (Paste)
  - 🔄 Alt+Tab (перемикання вікон)

### 🎵 Медіа:
- ⏮️ Попередній трек
- ⏯️ Play/Pause
- ⏭️ Наступний трек
- 🔊/🔉 Гучність +/-
- 🔇 Mute

### ⚙️ Система:
- 🚀 **Робота**: Вмикає екран, яскравість на макс
- 🌙 **Сон**: Вимикає екран
- 🔒 **Блокування**: Блокує ПК
- 💻 **Термінал**: Відкриває термінал
- 🛑 **Вимкнути**: Вимикає ПК (через 1 хв)
- 🔄 **Перезавантажити**: Перезавантажує ПК

## Доступ з інтернету (опціонально):

Якщо хочете керувати ПК з будь-якої точки світу:

### Варіант 1: Ngrok (найпростіший)
```bash
# Встановіть ngrok
sudo dnf install ngrok -y

# Запустіть тунель
ngrok http 5000
```

Скопіюйте URL (наприклад: `https://abc123.ngrok.io`) і оновіть `WEB_APP_URL` в `.env`

### Варіант 2: Cloudflare Tunnel
```bash
# Встановіть cloudflared
sudo dnf install cloudflared -y

# Створіть тунель
cloudflared tunnel --url http://localhost:5000
```

## Безпека:

⚠️ **ВАЖЛИВО**:
- Змініть `PC_PASSWORD` в `.env` на складний пароль
- Не діліться посиланням на веб-інтерфейс з незнайомими
- При використанні через інтернет - використовуйте HTTPS (ngrok/cloudflare автоматично)
- Rate limiting вже налаштований (30 запитів/хвилину)

## Troubleshooting:

### Трансляція не працює:
```bash
# Перевірте, чи встановлений FFmpeg
ffmpeg -version

# Перевірте, чи працює X11
echo $DISPLAY

# Спробуйте вручну
ffmpeg -f x11grab -video_size 1920x1080 -i :0.0 -f mjpeg -
```

### Миша/клавіатура не працює:
```bash
# Перевірте xdotool
xdotool --version

# Спробуйте вручну
xdotool mousemove 100 100
xdotool key space
```

### Порт 5000 зайнятий:
```bash
# Знайдіть процес
sudo lsof -i :5000

# Або змініть порт в app.py (останній рядок)
app.run(host='0.0.0.0', port=5001, threaded=True)
```

## API Endpoints:

- `GET /api/pc/screen_stream` - Трансляція екрану
- `POST /api/pc/screen_stream/stop` - Зупинити трансляцію
- `POST /api/pc/mouse` - Керування мишею
- `POST /api/pc/keyboard` - Керування клавіатурою
- `POST /api/pc/media` - Медіа контроль
- `POST /api/pc/scenario` - Сценарії (work/sleep)
- `POST /api/pc/power` - Живлення (shutdown/reboot)
