# 📋 Інструкція для сайту благодійних фондів

## 🚀 Як запустити сайт

### Варіант 1: Тільки локально (на вашому комп'ютері)
```bash
./start_charity.sh
```
або
```bash
python charity_server.py
```

Сайт буде доступний тільки на вашому комп'ютері: **http://localhost:5001**

---

### Варіант 2: З публічним доступом (для всіх через інтернет)
```bash
./start_charity_public.sh
```

Ви отримаєте:
- 🌐 Локальний доступ: http://localhost:5001
- 📱 Публічне посилання (наприклад: https://abc123.ngrok.io)

**Публічне посилання можна ділитися з ким завгодно!**

---

### Варіант 3: Разом з усіма ботами
```bash
python run_all.py
```

Запустить:
- API Сервер
- Charity Сервер (з публічним доступом)
- Telegram боти
- Ngrok тунелі

---

## 📝 Як користуватися сайтом

### Головна сторінка
- **Ліворуч**: список ресурсів (продукти, медикаменти, одяг тощо)
- **Праворуч**: картки з фондами для обраного ресурсу

### 📊 Сортування ресурсів
У лівій панелі є випадаюче меню з варіантами:
- **Своє налаштування** - перетягуйте ресурси мишкою (⋮⋮ іконка)
- **А → Я** - алфавітне сортування
- **Я → А** - зворотне алфавітне
- **Від новіших** - спочатку нові
- **Від старіших** - спочатку старі

### ➕ Додати новий ресурс
1. Натисніть кнопку **"➕ Додати ресурс"** у лівій панелі
2. Заповніть форму:
   - **Назва ресурсу**: наприклад, "Дитяче харчування"
   - **Опис**: наприклад, "Суміші, каші для немовлят"
   - **Іконка**: emoji символ, наприклад 🍼
3. Натисніть **"Зберегти"**

### ✏️ Редагувати ресурс
- На картці ресурсу натисніть іконку **✏️**
- Змініть потрібні дані
- Натисніть **"Зберегти"**

### 🗑️ Видалити ресурс
- На картці ресурсу натисніть іконку **🗑️**
- Підтвердіть видалення
- ⚠️ **УВАГА**: Всі фонди цього ресурсу також будуть видалені!

---

### ➕ Додати фонд до ресурсу
1. Натисніть на ресурс у лівій панелі
2. Натисніть кнопку **"➕ Додати фонд"** вгорі праворуч
3. Заповніть форму:
   - **Назва фонду**: наприклад, "Червоний Хрест України"
   - **Контакти**: телефон, сайт, email
   - **Умови надання**: хто може отримати допомогу, які документи потрібні
4. Натисніть **"Зберегти"**

### ✏️ Редагувати фонд
- На картці фонду натисніть іконку **✏️**
- Змініть дані
- Натисніть **"Зберегти"**

### 🗑️ Видалити фонд
- На картці фонду натисніть іконку **🗑️**
- Підтвердіть видалення

---

## 💾 Резервне копіювання даних

### Зберегти дані (експорт)
1. Натисніть кнопку **"💾 Експорт"** вгорі праворуч
2. Файл `charity_backup_YYYY-MM-DD.json` буде завантажено на ваш комп'ютер
3. Зберігайте цей файл у безпечному місці!

## ⚠️ Важливо!

1. **Регулярно робіть резервні копії** (кнопка "💾 Експорт")
2. **Не закривайте термінал** з запущеним сервером, поки працюєте з сайтом
3. **Перевіряйте дані** перед видаленням ресурсів
4. **Публічне посилання** змінюється при кожному перезапуску `start_charity_public.sh`
5. **Власний порядок** ресурсів зберігається в браузері кожного користувача окремо

3. Подивіться на помилки у терміналі
