## 🚀 Перший раз: Встановлення

### Крок 1: Завантажте проєкт

1. Відкрийте в браузері: `https://github.com/ваш-username/charity-funds`
2. Натисніть зелену кнопку **"Code"**
3. Виберіть **"Download ZIP"**
4. Збережіть файл на комп'ютер

### Крок 2: Розпакуйте

1. Знайдіть завантажений файл `charity-funds-main.zip`
2. Клік правою кнопкою → **"Розпакувати все"**
3. Виберіть зручне місце, наприклад: `C:\charity\`

### Крок 3: Встановіть Python (якщо ще не встановлено)

1. Завантажте з: https://www.python.org/downloads/
2. **ВАЖЛИВО:** При встановленні поставте галочку **"Add Python to PATH"**
3. Натисніть "Install Now"
4. Перезавантажте комп'ютер

### Крок 4: Запустіть сайт

1. Відкрийте папку з проєктом
2. Подвійний клік на **`start_charity_windows.bat`**
3. Відкриється чорне вікно - НЕ ЗАКРИВАЙТЕ його!
4. Відкрийте браузер: http://localhost:5001

**Готово!** Тепер можна додавати фонди та ресурси.

## 🔄 Як отримати оновлення (коли я щось змінив)

### Спосіб 1: Простий (рекомендую)

**Коли я скажу "Оновлення доступне":**

**Готово!** Тепер у вас нова версія з усіма вашими даними.

## 💾 Резервне копіювання (дуже важливо!)

### Автоматичне (рекомендую)

Створіть файл `auto_backup.bat` в папці проєкту:

```batch
@echo off
set BACKUP_DIR=C:\charity\backups
set DATE=%date:~-4%%date:~3,2%%date:~0,2%

if not exist "%BACKUP_DIR%" mkdir "%BACKUP_DIR%"
copy charity_funds.db "%BACKUP_DIR%\backup_%DATE%.db"
echo ✅ Резервна копія створена!
pause
```

Запускайте його раз на тиждень або перед оновленням.

### Ручне

Просто скопіюйте файл `charity_funds.db` в іншу папку.

## 🆘 Що робити якщо щось пішло не так

### Проблема: "Всі дані зникли!"

**Рішення:**
1. Знайдіть резервну копію `charity_funds.db`
2. Скопіюйте її в папку з проєктом
3. Перезапустіть сервер

### Проблема: "Сайт не запускається після оновлення"

**Рішення:**
1. Перевірте що файл `charity_funds.db` на місці
2. Видаліть папку `__pycache__` (якщо є)
3. Перезавантажте комп'ютер
4. Спробуйте знову

### Проблема: "Python не знайдено"

**Рішення:**
1. Переконайтеся що Python встановлено
2. При встановленні поставте галочку "Add Python to PATH"
3. Перезавантажте комп'ютер
