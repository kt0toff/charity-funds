# ⚡ ШВИДКИЙ СТАРТ - 5 ХВИЛИН

## 🎯 ЩО ТОБІ ПОТРІБНО ЗРОБИТИ

### 1️⃣ **Встанови залежності (2 хвилини)**

```bash
pip install -r requirements.txt
```

### 2️⃣ **Запусти Бекенд (1 хвилина)**

```bash
python app.py
```

Побачиш:
```
🚀 COUPLE BOT API ЗАПУЩЕНО!
📡 Сервер: http://localhost:5000
```

### 3️⃣ **У НОВОМУ ТЕРМІНАЛІ запусти Бота (1 хвилина)**

Спочатку **замініть токен** в `telegram_bot_couples.py`:

```python
BOT_TOKEN = "ТВІЙ_ТОКЕН_ТУТ"  # ← ЗАМІСТЬ ЦЬОГО ВСТАВИТЬ ТОКЕН ВІД @BotFather
```

Потім запусти:

```bash
python telegram_bot_couples.py
```

Побачиш:
```
✅ База даних ініціалізована!
✅ Команди встановлено в меню BotFather!
✅ Бот запущено!
```

### 4️⃣ **Запусти фронтенд веб-сервер (1 хвилина)**

```bash
python -m http.server 8000
```

## ❓ ЩО РОБИТИ ЯКЩО ЧОС НЕ ПРАЦЮЄ?

### Ошибка: "ModuleNotFoundError"

```bash
pip install -r requirements.txt
```

### Ошибка: "Port 5000 is already in use"

Змінити порт в `app.py`:
```python
PORT = 5000  # ← змінити на 5001, 5002 тощо
```

### Ошибка: "Database is locked"

Закрити SQLite Browser або переза запустити Flask

### Mini App не відкривається

- Перевір чи запущено веб-сервер на :8000
- Перевір чи вірний URL в `main_keyboard()`
