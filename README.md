# Charity Funds Directory 🤝

Веб-сайт для управління інформацією про благодійні фонди та ресурси, які вони надають.

## 🌟 Особливості

- 📋 Список ресурсів з фільтрацією та сортуванням
- 🎯 Картки фондів з детальною інформацією
- ✏️ Повне редагування (додавання, зміна, видалення)
- 💾 Автоматичне збереження в GitHub
- 📱 Адаптивний дизайн
- 🌐 Безкоштовний хостинг на Netlify

## 🚀 Деплой на Netlify

### 1. Створіть GitHub Personal Access Token

1. Перейдіть на https://github.com/settings/tokens
2. Натисніть "Generate new token" → "Generate new token (classic)"
3. Назва: `charity-funds-netlify`
4. Виберіть права:
   - ✅ `repo` (повний доступ до репозиторіїв)
5. Натисніть "Generate token"
6. **ВАЖЛИВО:** Скопіюйте токен (він показується тільки один раз!)

### 2. Задеплойте на Netlify

1. Зареєструйтесь на https://netlify.com через GitHub
2. Натисніть "Add new site" → "Import an existing project"
3. Виберіть "Deploy with GitHub"
4. Знайдіть репозиторій `charity-funds`
5. Натисніть "Deploy site"

### 3. Налаштуйте Environment Variables

1. В Netlify Dashboard перейдіть в **"Site settings"** → **"Environment variables"**
2. Додайте змінні:
   - `GITHUB_TOKEN` = ваш токен з кроку 1
   - `GITHUB_OWNER` = `kt0toff` (ваш GitHub username)
   - `GITHUB_REPO` = `charity-funds`
   - `GITHUB_BRANCH` = `main`
3. Натисніть "Save"
4. Перейдіть в **"Deploys"** → **"Trigger deploy"** → **"Deploy site"**

### 4. Готово! 🎉

Сайт буде доступний за адресою `your-site.netlify.app`

## 📝 Як користуватись

1. Відкрийте сайт
2. Натисніть "➕ Додати ресурс" щоб додати категорію (наприклад "Продукти харчування")
3. Оберіть ресурс зі списку
4. Натисніть "➕ Додати фонд" щоб додати благодійний фонд
5. Всі зміни автоматично зберігаються в GitHub
6. Netlify автоматично оновлює сайт після кожної зміни (1-2 хвилини)

## 🆘 Troubleshooting

### Помилка "GitHub token not configured"
- Перевірте, чи додали `GITHUB_TOKEN` в Environment Variables
- Перевірте, чи токен має права `repo`
- Зробіть Trigger deploy після додавання змінних

### Зміни не зберігаються
- Перевірте консоль браузера (F12) на помилки
- Перевірте, чи всі Environment Variables налаштовані правильно
- Перевірте, чи токен не застарів

### Сайт не оновлюється після змін
- Netlify автоматично оновлює сайт після коміту в GitHub
- Зачекайте 1-2 хвилини
- Очистіть кеш браузера (Ctrl+Shift+R)
