# Довідник благодійних фондів

Статичний сайт з адмін-панеллю для керування благодійними фондами.

## 🚀 Швидкий старт

### Деплой на Netlify (БЕЗКОШТОВНО)

1. **Зареєструйтесь на Netlify:**
   - Перейдіть на https://netlify.com
   - Натисніть "Sign up" → "Sign up with GitHub"
   - Авторизуйтесь через GitHub

2. **Задеплойте сайт:**
   - Натисніть "Add new site" → "Import an existing project"
   - Виберіть "Deploy with GitHub"
   - Знайдіть репозиторій `charity-funds`
   - Натисніть "Deploy"
   - Готово! Сайт буде доступний за адресою типу `your-site.netlify.app`

3. **Налаштуйте адмін-панель:**
   - В Netlify Dashboard перейдіть в "Site settings" → "Identity"
   - Натисніть "Enable Identity"
   - Перейдіть в "Registration" → виберіть "Invite only"
   - Перейдіть в "Services" → "Git Gateway" → "Enable Git Gateway"
   - Перейдіть в "Identity" → "Invite users"
   - Введіть email мами і надішліть запрошення

4. **Перший вхід:**
   - Мама отримає email з посиланням
   - Натисне на посилання і створить пароль
   - Зайде на `your-site.netlify.app/admin/`
   - Готово! Можна редагувати контент

## 📝 Як користуватись адмін-панеллю

1. Зайдіть на `your-site.netlify.app/admin/`
2. Введіть email і пароль
3. Виберіть "Благодійні фонди" або "Налаштування сайту"
4. Натисніть "Редагувати" або "Новий запис"
5. Внесіть зміни
6. Натисніть "Зберегти"
7. Зміни з'являться на сайті через 1-2 хвилини

## 📂 Структура проекту

```
charity-funds/
├── index.html              # Головна сторінка сайту
├── admin/
│   ├── index.html         # Адмін-панель
│   └── config.yml         # Конфігурація CMS
├── content/

│   └── settings.json      # Налаштування сайту

└── netlify.toml          # Конфігурація Netlify

```
